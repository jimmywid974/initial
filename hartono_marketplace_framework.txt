>>>>>>>>> Project Marketplace

Mendevelop aplikasi untuk mengintegrasikan marketplace yang dimiliki Hartono (Shopee, Tokopedia, dll) supaya dapat dikelola satu pintu yaitu lewat aplikasi tersebut. 

Semua proses yang terjadi di marketplace, mulai dari menerima pesanan, pengecekan tock, penjadwalan pengiriman dari Gudang terdekat dari tempat tujuan pengiriman, pembuatan sales order dan faktur, serta retur penjualan dapat dilakukan lewat aplikasi tersebut pada semua marketplace yang dimiliki Hartono.

Bagian Frontend dibuat dengan Bahasa PHP dengan framework Laravel dan bagian Backend dipanggil menggunakan API yang menggunakan Bahasa pemrograman C# dengan framework .NET Core. DBMS yang digunakan ada 2 yaitu Microsoft SQL Server dan PostgreSQL
